# Jantar_dos_Filosofos
O clássico problema de Sistemas Operacionais, Jantar dos Filósofos, em três versões: Uma que causa deadlock, outra corrigida com ordenação de recursos (filósofo ímpar pega garfo direito primeiro) e uma última versão corrigido usando semáforo limitante (no máximo N-1 filósofos pegam garfos simultaneamente).
